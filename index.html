"¿Cómo sacas el inventario de hardware de 100 PCs antes de las 12:00?"

Opción A: El Becario

Va PC por PC con un pendrive. Tarda 1 semana y comete errores.

DESPEDIDO
Opción B: El Senior

Escribe un script maestro. Tarda 1 minuto y es perfecto.

ASCENDIDO

Ejecuta este comando una sola vez en tu consola antes de lanzar el script:

Set-ExecutionPolicy RemoteSigned -Scope Process
Auditor_Hardware.ps1
# 1. Seguridad: Mover al escritorio
cd $HOME\Desktop

# 2. Técnica del Sándwich (Agrupar datos)
$(
    "--- CPU ---" 
    Get-CimInstance Win32_Processor | Select-Object Name, NumberOfCores | Out-String

    "--- RAM ---"
    Get-CimInstance Win32_PhysicalMemory | Select-Object Capacity, Speed | Out-String

    "--- DISCO ---"
    Get-PhysicalDisk | Select-Object FriendlyName, MediaType, Size | Out-String

) | Out-File "INFORME_JEFE.txt"

# 3. Resultado
notepad.exe INFORME_JEFE.txt

Entendiendo el Código (Paso a Paso)

0. Permisos (Vital)
Set-ExecutionPolicy RemoteSigned ...
  • Set-ExecutionPolicy: Pide permiso a Windows para usar scripts.
  • -Scope Process: El permiso solo dura mientras tengas abierta ESTA ventana (Modo seguro).
1. Preparación
cd $HOME\Desktop
  • cd: Comando para viajar por carpetas.
  • $HOME: Variable automática que lleva a tu escritorio.
  • Objetivo: Que el informe aparezca delante de tus narices.
2. Técnica del Sándwich
$( ... ) | Out-File "INFORME.txt"
  • $( ): La "Bolsa Mágica". Agrupa todo lo que ocurre dentro.
  • Get-CimInstance: Interroga a Windows (CPU, RAM).
  • | Out-File: Envía la bolsa cerrada al archivo de texto.
3. El Resultado
notepad.exe INFORME_JEFE.txt
  • notepad.exe: Usamos .exe para evitar conflictos de sistema.
  • Resultado: El informe se abre automáticamente al terminar.

🚀 Auditoría Masiva (God Mode)

Este es el contenido para el archivo Auditor_Remoto.ps1: 

# ==========================================
#  AUDITOR REMOTO (MODO SILENCIOSO)
# ==========================================
$(
    "--- REPORTE DESDE: $env:COMPUTERNAME ---"

    "--- CPU ---" 
    Get-CimInstance Win32_Processor | Select-Object Name, NumberOfCores | Out-String

    "--- RAM ---"
    Get-CimInstance Win32_PhysicalMemory | Select-Object Capacity, Speed | Out-String

    "--- DISCO ---"
    Get-PhysicalDisk | Select-Object FriendlyName, MediaType, Size | Out-String
)
📂 Cómo se ejecuta (El "Comando Maestro")
Invoke-Command -ComputerName (Get-Content lista_pcs.txt) -FilePath ".\Auditor_Remoto.ps1" | Out-File "$HOME\Desktop\REPORTE_GLOBAL.txt"

Al terminar, abres el informe centralizado:
notepad.exe "$HOME\Desktop\REPORTE_GLOBAL.txt"
