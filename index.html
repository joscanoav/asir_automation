/// Lunes, 8:00 AM ///

El Jefe de Sistemas entra por la puerta con una carpeta.

"¿Cómo sacas el inventario de hardware de 100 PCs antes de las 12:00?"

Opción A: El Becario

Va PC por PC con un USB ejecutando dxdiag y apuntando en un Excel. Tarda 1 semana. Errores humanos garantizados.

DESPEDIDO
Opción B: El Senior

Se sienta, abre PowerShell ISE. Escribe un script de 10 líneas. Ejecuta. Se toma un café.

ASCENDIDO
Administrator: Windows PowerShell

# 0. Permisos (Vital si te sale error rojo)
Set-ExecutionPolicy RemoteSigned -Scope Process

# 1. Seguridad: Mover al escritorio
cd $HOME\Desktop

# 2. Técnica del Sándwich (Agrupar datos)
$(
    "--- CPU ---" 
    Get-CimInstance Win32_Processor | Select-Object Name, NumberOfCores

    "--- RAM ---"
    Get-CimInstance Win32_PhysicalMemory | Select-Object Capacity, Speed

    "--- DISCO ---"
    Get-PhysicalDisk | Select-Object FriendlyName, MediaType, Size

) | Out-File "INFORME_JEFE.txt"

# 3. Resultado (¡PON .EXE PARA QUE NO SALGA EL RARO!)
notepad.exe INFORME_JEFE.txt

Entendiendo el Código (Paso a Paso)

0. Permisos (Vital)
Set-ExecutionPolicy RemoteSigned ...
  • Set-ExecutionPolicy: Pide permiso a Windows para usar scripts.
  • -Scope Process: El permiso solo dura mientras tengas abierta ESTA ventana (Modo seguro).
1. Preparación
cd $HOME\Desktop
  • cd: Comando para viajar por carpetas.
  • $HOME: Variable automática que lleva a tu carpeta personal.
  • Objetivo: Que el informe aparezca delante de tus narices.
2. Técnica del Sándwich
$( ... ) | Out-File "INFORME.txt"
  • $( ): La "Bolsa Mágica". Agrupa todo lo que ocurre dentro.
  • Get-CimInstance: Interroga a Windows (CPU, RAM).
  • | Out-File: Envía la bolsa cerrada al archivo de texto.
3. El Resultado
notepad.exe INFORME_JEFE.txt
  • notepad.exe: Usamos .exe para evitar conflictos con Git.
  • Resultado: El informe se abre automáticamente al terminar.

¿Y para los 100 PCs?

No tienes que levantarte de la silla. Despliega el poder de Remote Management.

Invoke-Command -ComputerName (Get-Content lista_pcs.txt) -FilePath "Auditor_Hardware.ps1"
